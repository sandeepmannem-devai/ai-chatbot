# 🤖 AI Chatbot

This is a simple AI chatbot built using Python, Streamlit, and OpenAI API.

## 🚀 Features
- Ask questions and get AI responses
- Simple UI using Streamlit
- Powered by OpenAI

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenAI API

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
